# Linux Automation and Log Analysis Scripts

This repository contains a collection of Linux shell scripting tasks focused on system monitoring, log analysis, and automation. These scripts demonstrate practical Linux skills used in backend engineering, system administration, and cybersecurity environments.

## Overview

The project includes scripts that automate common administrative tasks such as:

- Monitoring system boot history
- Extracting usernames from system files
- Analyzing error logs
- Displaying dynamic system information
- Logging user session activity

These tasks are implemented using Bash scripting and standard Linux utilities.

---

## Tasks Included

### 1. Boot History Analyzer
Extracts and displays system boot history using Linux log files.

**Concepts used:**

- last command
- log parsing
- system monitoring

---

### 2. Error Log Grepper

Searches and extracts specific error messages from log files.

**Concepts used:**

- grep
- pattern matching
- log filtering

---

### 3. Username Extractor

Extracts usernames from system files such as `/etc/passwd`.

**Concepts used:**

- awk
- cut
- text processing

---

### 4. Dynamic MOTD Generator

Displays dynamic system information such as:

- Logged-in users
- System uptime
- System information

**Concepts used:**

- system commands
- automation
- shell scripting

---

### 5. User Session Logger

Logs and tracks user login sessions.

**Concepts used:**

- who command
- session monitoring
- logging
